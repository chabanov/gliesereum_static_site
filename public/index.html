What Blockchain Is?

At its core blockchain is a database distributed in a network with multiple independent p2p servers (nodes) that constantly synchronize information with each other. Each node contains exactly the same copy of a growing set of data blocks linked to each other using cryptographic hash. Blocks of digital information linked using a cryptographic hash of the previous block are being appended to a growing chain when new transactions appear in the network. Unique hash fingerprints, network consensus with regards to the stored data, and network decentralization, together guarantee security and immutability of an information in the database. Distributed structure of a network removes a lack of trust associated with a storage of information in one place only.

How Gliesereum Works

Gliesereum provides a lightweight distributed blockchain database structure for small and large corporate entities using modest hardware resources via a p2p network with independent servers, each server node containing identical copies of the database. These nodes are called Gliesereum Core Nodes (GCNs) or master nodes. Master node is THE node needed to become a member of the network. Gliesereum Network Node is a second ‘helper’ type of node with limited functionality that can be added to the corporate subnetworks to speed up transaction processing.
X13 is a “chip”network trust protocol that contains our new Proof-of-Legend (PoL) consensus algorithm. It is a lightweight protocol that is at the core of the calculation speed up across the network. The protocol is used to create Smart Transactions (a next generation analog of Smart Contracts), sign and validate any documents and tokens that migrate in the network, including the encrypted highly secure PUT objects, and calculate consensus between the master nodes. X13 can be incorporated into external blockchain networks to replace original consensus protocols without re-build of original database structures. Three integration scenarios are possible:

Default System Tokens

PUT token is a highly secure network object that can travel across the network and can be opened or modified by the owners of its private keys only. Examples: contracts, passports, confidential documents, jurisdictional court decisions, etc.

Native token GRAPH is a main token in the ecosystem, emitted to GCNs that validate new transactions, issued as a payment instrument for transaction processing, or can be given as a reward via DApp operation.
